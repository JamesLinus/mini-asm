# mini-asm
Custom mini ASM language &amp; CPU emulator

***Work in progress!***

This is a mini ASM language based on 6502 and conventional instructions (MOV, etc...).

#Registers
This mini ASM provides some registers:
* A (Accumulator)
* X (X index)
* Y (Y index)
* P (Program status)
* PC (Program Counter)
* S (Stack pointer)
